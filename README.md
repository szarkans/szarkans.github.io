# Официальная Вики Кошкокрафта

Привет! Это репозиторий Вики сервера [Кошкокрафт](https://discord.gg/6f3FwFRJWC).

Вики доступна по ссылке <https://wiki.catcraft.ru>

---

## Что нужно для запуска

Перед началом установи [Node.js](https://nodejs.org/) версии **18 или новее**.

Проверить, что всё установилось, можно командами в терминале:

```bash
node --version   # должно показать v18+ (например v22.x)
npm --version
```

---

## Как запустить сайт локально

1. Скачай (клонируй) репозиторий и зайди в его папку:

   ```bash
   git clone https://github.com/szarkans/szarkans.github.io.git
   cd szarkans.github.io
   ```

2. Установи зависимости (это нужно сделать один раз, потом — только если они обновились):

   ```bash
   npm install
   ```

3. Запусти дев-сервер:

   ```bash
   npm run docs:dev
   ```

4. Открой в браузере адрес, который покажет терминал, обычно <http://localhost:5173>.

Чтобы остановить сервер, нажми в терминале `Ctrl + C`.

### Остальные команды

| Команда | Что делает |
| --- | --- |
| `npm run docs:dev` | Запуск дев-сервера с авто-обновлением |
| `npm run docs:build` | Собрать готовую (продакшен) версию сайта |
| `npm run docs:preview` | Посмотреть собранную версию локально |
